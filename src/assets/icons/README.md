# Book Icon Assets

Included files:

- `book-icon.svg`: primary polished app icon with gradients and depth
- `book-icon-flat.svg`: simpler flat variant for Android adaptive icons or small UI surfaces
- `book-icon-monochrome.svg`: neutral monochrome version for badges, settings, and dark/light system contexts

Design notes:

- warm amber background for an energetic study feel
- open-book silhouette to signal reading and learning
- bookmark accent to make the mark feel app-like rather than generic

If you want PNG exports later, these SVGs are already sized on a `1024x1024` artboard.
